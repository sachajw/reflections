Question
What happens when you initialize a repository? Why do you need to do it?
Answer
It creates the hidden git folder to start tracking the project files therein. It creats a replica of the repository
to your local hard drive so that you can work online or offline.

Question

Answer
It gives you flexibility when adding files as logical commits

Question
How can you use the staging area to make sure you have one commit per logical change?
Answer
By specifying a seperate git add command per change.
